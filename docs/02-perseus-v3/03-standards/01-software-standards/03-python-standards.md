# Python

Just follow the conventions in [PEP 8](https://peps.python.org/pep-0008). A couple of the most important notes are:

- Classes: `PascalCase`
- Variables: `snake_case` variable names
- 4 space indentation

Everything in the [general standards](01-general-standards.md) section still applies, and if there is a conflict between PEP 8 and what's specified in these docs, these docs take priority.

## Formatting

### Formatter
