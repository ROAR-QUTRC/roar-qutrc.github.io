Namespace hi_can::addressing::drive::vesc
=========================================


Namespace containing all addresses in the VESC subsystem. 




Detailed Description
--------------------

Note that VESCs kinda have to be shoehorned into the Hi-CAN spec, so their address isn't *entirely* compliant with the standard. By treating them as "system 0, subsystem 0" devices, we can isolate the issues and make it mostly work
