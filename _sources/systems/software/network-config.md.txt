# Network Configuration

To configure Unifi firmware, connect to the network (likely by ethernet at first, but after it's set up, wifi will work), then go to https://unifi/network/default/dashboard.

## UCG Max

No configuration necessary.

## U7 Outdoor

Needs to be configured as a 'Mesh Parent' as well as to allow 'Mesh Connect'.
Needs to have the correct antenna configuration selected - either 'Omni Antenna' (less range, wide angle) or 'Built-in Antenna' (higher range, less angle).

## U7 Long Range

Needs to be configured to allow 'Mech Connect'.

## US 8 PoE

No configuration necessary.

## Network

The network name (SSID) should be "QUTRC-ROAR-LOCAL" and the password should be shared with all the leads.
