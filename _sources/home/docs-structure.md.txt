## Tooling

This documentation is built using [Sphinx](https://www.sphinx-doc.org/en/master/) with the [MyST](https://myst-parser.readthedocs.io/en/latest/) parser to add markdown support. Code documentation is provided by [Doxygen](https://www.doxygen.nl/), which is linked to Sphinx with the [Breathe](https://breathe.readthedocs.io/en/latest/) extension, along with [Exhale](https://exhale.readthedocs.io/en/latest/) to add Class, File, and Page hierarchies.
